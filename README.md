# Basic-Line-Tracer-using-TT-Motors-
Made this Line tracer for the Robosports SummerCup (RSC1) VisMin Leg Competition. We had no line tracers at the time so i had to make one myself. I Came up with this design and got 4th place. Not bad for a national level competition.

## Components
- Arduino 
- 5 Channel ir sensor
- TB661FNG motor driver
- 2 DC TT motors
- 2 Li-ion 3.7v battery

## How it Works
The IR sensors detect the line.
A PID controller calculates the error.
Motor speeds are adjusted to keep the robot centered.
